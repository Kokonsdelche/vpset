## نصب و راه‌اندازی

1. کلون کردن مخزن:
```bash
git clone https://github.com/your-username/your-repo-name.git
```

2. نصب وابستگی‌ها:
```bash
npm install
# یا
yarn install
```

3. اجرای پروژه در محیط توسعه:
```bash
npm run dev
# یا
yarn dev
```

4. ساخت نسخه تولید:
```bash
npm run build
# یا
yarn build
```
